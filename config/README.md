# Configurações do Sistema

Esta pasta contém os arquivos de configuração do sistema.

## Estrutura

- `templates.json` - Arquivo de configuração exportado com templates e configurações
- `images/` - Pasta para armazenar imagens dos cabeçalhos
  - `cabecalho_voluntaria.png` - Imagem do cabeçalho para internação voluntária
  - `cabecalho_involuntaria.png` - Imagem do cabeçalho para internação involuntária

## Como usar

1. Exporte as configurações através do botão "Exportar Config" na tela de configurações
2. Coloque o arquivo `templates.json` nesta pasta
3. Exporte as imagens através do botão "Exportar Imagens"
4. Coloque as imagens exportadas na pasta `images/` desta pasta
